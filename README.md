# L63 Media
This website is portfolio website for L63 Media which showcases the companies work

## Technologies
- Nextjs
- Typescript
- Tailwind CSS
- GSAP

## Features
- Hover animations with GSAP and CSS
- Contact form with ratelimiting logic and Resend
- Responsive design

## The build process
This is one of my more creative projects that I built for my friends company L63 Media. It is designed by me in cooperation with my friend Adam.
